Git
###

* For Windows, there are two ways to install Git. Go to the `website <http://git-scm.com/download/win>`_ and download
will start automatically. Another way to install Git is to install Github for Windows, go to the next section. You can
download it from `Github <http://windows.github.com>`_.

* For Linux, go to your user folder on your local machine and open a new terminal.

.. code-block:: python
    sudo apt install git-all

For a specific version, installation instructions are available on the `Git website <http://git-scm.com/download/linux>`_.

Github
######

Github allows collaborative coding and automating all your software development workflows. This tool is the largest
development platform in the world.

* Create an account on the `website <https://github.com/join?ref_cta=Sign+up&ref_loc=header+logged+out&ref_page=%2F&source=header-home>`_ to access this tool.
You can collaborate with your team on different projects thanks to it.

* This software only can be download on Windows and macOS. It will be available on your `Desktop <https://desktop.github.com/>`_. Then, sign in.

You can learn how to use it with tutorials on Youtube or other websites. The most complete documentation is available on
Github, just select the `section <https://docs.github.com/en/github>`_ you need.

Gitkraken
#########

* Download on this `tab <https://www.gitkraken.com/download>`_.Then, install it. This software is available for all
platforms. Once it is done, the platform offers multiple `tutorials <https://www.gitkraken.com/learn/git/tutorials>`_ with different levels, select the one you are
